Food Delivery Application (Dockerized)

A full-stack Food Delivery application with frontend, backend, and PostgreSQL database, fully containerized using Docker and orchestrated with Docker Compose. This project demonstrates a real-world setup for deploying multi-service applications in containers.

Features

- Frontend: React.js application displaying a dynamic menu of food items.
- Backend: Flask API serving food data and connecting to PostgreSQL.
- Database: PostgreSQL storing food menu items.
- Dockerized: Frontend, backend, and database run in isolated containers.
- Ready-to-run: Just `docker-compose up` to start the full application.
